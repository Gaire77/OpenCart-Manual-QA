# OpenCart Manual QA Project

## Project Overview
This project demonstrates **manual QA testing** of the OpenCart e-commerce website. The testing covers major functionalities such as login, search, product details, cart, checkout, and UI validation.

## Project Deliverables
- **Test Cases:** `TestCases/TestCases.xlsx`
- **Bug Reports:** `BugReports/BugReport.xlsx`
- **Test Plan:** `Documentation/TestPlan.docx`
- **Test Summary:** `Documentation/TestSummary.docx`
- **Screenshots:** `Screenshots/` (evidence of testing)

## Tools Used
- Microsoft Excel
- Microsoft Word
- Browsers: Chrome, Edge, Firefox
- Jira (simulated for bug tracking)

## Features Tested
- User Login / Logout
- Product Search
- Product Details Page
- Add to Cart
- Checkout Process
- UI and Responsiveness

## How to Showcase
1. Open **TestCases.xlsx** to see all test cases executed with status and actual results.
2. Open **BugReport.xlsx** to see all identified bugs and their severity.
3. Open **TestPlan.docx** to see project scope, objectives, and testing methodology.
4. Open **TestSummary.docx** for a summary of all executed test cases and findings.
5. View **Screenshots/** folder to see visual evidence of testing.
